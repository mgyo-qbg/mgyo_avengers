## 화면 구성 (80개 이상 페이지)

### 어드바이저 대시보드 (`/dashboard/advisor/*`)
- 메인 대시보드, 고객목록/등록(상세/간편)/체험고객
- 투자모델: 자산배분유형, QB포트폴리오, 인출전략모델
- 업무관리(FA_ADMIN): FA 생성/관리, 포트폴리오 관리
- 자녀 유학자금: 입력폼, 결과
- 마켓 콤파스(미사용), 알림(비활성), 설정, 이용안내

### 고객 대시보드 (`/dashboard/client/*`)
- 블루프린트 (자산/목표/현금흐름 탭)
- 프로필: 기본정보, 재무목표, 자산, 소득, 지출, 저축 (6 서브페이지)
- 은퇴설계: 현금흐름, 은퇴분석, 국민연금, 연금인출전략, 3층 연금분석
- 투자설계: 투자진단, 투자제안, 종목진단
- 헤지도구: 소진시점 계산기, 목표기반투자, 투자분석기, 수수료 평가
- 도구함: 상속, 투자심리분석
- 리포트: 맞춤/심층

### 외부 페이지 (`/external/*`)
- 본인인증(Go/Complete), 투자성향진단, 약관

### PDF (`/pdf/*`)
- 커버, 목차, 컴플라이언스, 첨부, 어드바이저 코멘트, 마지막 페이지
- 블루프린트, 은퇴분석, 현금흐름, 연금, 인출전략, 투자진단/제안/전략, 종목진단

## 인증/권한 플로우

1. 로그인 (`/auth/login`) → JWT 인증
2. 역할 기반 리다이렉트: FA/FA_ADMIN → 고객목록, ABROAD_GBI → 유학자금 폼
3. 약관 미동의 시 ConfirmTerm 모달
