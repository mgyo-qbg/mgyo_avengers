## 안건 및 논의 내용

### 1. 리포트 성격
- 정확한 수치 리포트가 아닌, 투자진단과 동일한 흐름으로 필요성을 공유하는 느낌

### 2. 국민연금 수령 나이
- **고객 출생연도 기반으로 넛지 API에서 일반수령 나이를 받아 적용** (하드코딩 없음)
- 국민연금 일반수령 기준은 **넛지 계산 활용** (설문 플로우부터 동일하게 적용)

### 3. [인출에 사용할 자산은 얼마나 있을까요?] 페이지
- 자산 구분을 **안전자산 / 위험자산** 두 가지로만 표시 (평가금액 기준)

### 4. [현재 자산배분은 인출 전략에 적합한가요?] 페이지
- 가이드라인 적용 시 **옵셔널 처리 고민 가능** (빼도 무방)
- 추가 논의 필요

### 5. 건보료 (지식도구함)
- 정보성 콘텐츠로 제공
- **예상 건보료 계산 기능은 현재 없음** → "직접 확인" 안내 방향으로 처리

### 6. [국민연금, 언제 받는 게 가장 유리할까요?] 페이지
- **연도별 수령액 누적 라인 차트 3개** 추가 가능성 (조기 / 일반 / 연기)
- 손익분기점 분석: 조기↔일반 역전 포인트, 조기↔연기 역전 포인트, 일반↔연기 역전 포인트
- 넛지 개발 필요 → **개발 지연 시 미포함 가능**

### 7. 문구
- 톤앤매너만 맞추는 방향 (원성님이 필요 문구 직접 작성)

### 8. [시뮬레이션 결과] 페이지
- 하단에 **"돈이 남아야 하는 이유"** 섹션 추가 (원성님 요청)
